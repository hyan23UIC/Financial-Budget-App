Overview

Financial Adventure is an interactive, story-driven budgeting game built using Python’s turtle graphics.
Players embark on a Pokémon-style adventure where they chase their financial goal, evolve through stages, battle enemies representing financial challenges, and finally receive personalized budgeting advice.

This project transforms personal finance concepts into a fun, animated experience where user inputs directly influence the game’s outcome.

Features
Goal Setting

Players begin by entering their financial goal in dollars.
The adventure adapts based on the player’s goals and financial situation.

Income & Expense Tracking

The game asks the user for:

- Monthly income
- Housing expenses
- Grocery expenses
- Transportation
- Debt payments
- Additional expenses

All values are validated to prevent invalid inputs.

Real-Time Budget Calculation

The game automatically computes:

- Total monthly expenses
- Remaining balance after expenses
- Months needed to achieve the user’s financial goal


Interactive Animation

The project includes:

- Turtle walking animations
- Character evolution sequences
- Enemy encounters
- A storyline that visually represents financial concepts


Guided Recommendations

Depending on the player's finances, the game provides:

- Success messages
- Number of months needed to reach the goal
- Budgeting tips (including the 50/30/20 rule)
- Warnings if expenses exceed income


Gameplay Flow

1. Start the game by clicking the “Start” button.
2. Choose your character.
3. Enter your financial goal.
4. Chase your goal through animated sequences.
5. Battle financial enemies like Income and Expenses.
6. Input monthly expenses in different categories.
7. Experience character evolutions based on progress.
8. Receive a detailed personalized financial summary.
9. End the game with recommendations or a victory message.



Technologies Used

- Python 3
- Turtle Graphics
- Custom GIF-based sprites:
   - Character sprites (Turtwig → Grotle → Torterra)
   - Enemy sprites (Empoleon, Emboar, etc.)
   - UI sprites (Next button, text box, backgrounds)
